## 前端 - 浏览器

### 独立安装

下载并且使用script标签引入,  `bearcat` 将以全局变量出现.

### NPM - 使用 [browserify](https://github.com/substack/node-browserify)

```bash
$ npm install bearcat
```

```js
var bearcat = require('bearcat');
```

### Bower

```bash
$ bower install bearcat
```
